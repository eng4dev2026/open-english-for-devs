# open-english-for-devs 🖥️

> A curated vocabulary list for IT professionals learning English in real-world tech contexts.
>
> 为 IT 从业者整理的英语词汇表，结合真实技术场景，附中文释义与例句。

---

## What is this? / 这是什么？

**open-english-for-devs** is an open-source vocabulary reference for developers, QA engineers, project managers, and anyone working in tech who wants to improve their English communication skills.

**open-english-for-devs** 是一个开源词汇参考项目，面向开发者、QA 工程师、项目经理等 IT 从业者，帮助他们提升英语沟通能力。

Each word or phrase includes:
每个词条包含：

- ✅ A clear Chinese explanation / 简明中文释义
- ✅ A real-world example sentence / 真实场景例句
- ✅ Chinese translation of the example / 例句中文翻译

---

## Who is this for? / 适合谁？

- 🧑‍💻 Developers reading English documentation / 阅读英文文档的开发者
- 🌍 Non-native English speakers working in international teams / 在国际团队工作的非母语英语使用者
- 🎓 Students preparing for IT careers / 备战 IT 职业的在校学生
- 📋 Anyone who wants to "speak like a dev" / 任何想说一口地道 IT 英语的人

---

## How to Use / 如何使用

1. Browse the folders / 浏览文件夹
2. Find the word list that matches your needs / 找到你需要的词汇表
3. Study the Chinese explanation and example sentence / 学习中文释义和例句
4. Practice using the word in your own sentences / 用这个词造句练习
