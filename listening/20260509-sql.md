## 📝 SQL 听力材料

---

**Hello, and welcome! Today, we are going to talk about SQL.**

SQL stands for "Structured Query Language." We use SQL to work with databases. A database is a place where we store information — like names, numbers, or dates.

You have probably heard of some databases before. MySQL, PostgreSQL, and Microsoft SQL Server are all very popular. They all use SQL.

SQL was created in the 1970s. That is more than 50 years ago! But people still use it every day. It is one of the most important tools in technology.

---

**So, what is a relational database?**

Think about an Excel spreadsheet. You have rows and columns. Each column has a type of information — for example, "Name" or "Age." Each row is one record — one person, one product, or one order.

Every row has a special ID number. We call this the **primary key**. It is like a name tag for each row — it is unique.

Now, we can connect two tables together. How? We take the ID from one table and put it in another table. This is called a **foreign key**.

For example: We have a **Teams** table and a **Players** table. In the Teams table, the Team ID is the primary key. In the Players table, the Team ID is the foreign key. This tells us: one player belongs to one team, and one team can have many players.

---

**Now, let's look at how we write SQL.**

An SQL statement is a piece of code that tells the database what to do. Let's look at the most common words.

The first word is **SELECT**. SELECT means "I want to see this information." For example: SELECT name, age — this means we want to see the name and the age.

The next word is **FROM**. FROM tells the database which table to look at. For example: FROM players — this means we are looking at the players table.

Then we have **WHERE**. WHERE adds a condition. For example: WHERE age is greater than 18 — this means we only want players who are older than 18. The database looks at every row and only returns the rows that match the condition.

Finally, we have **JOIN**. JOIN connects two tables together. We match the primary key in one table with the foreign key in another table. This is very powerful!

---

**Let's put it all together.**

Here is a simple SQL statement:

> *SELECT name, age FROM players WHERE age > 18 JOIN teams ON players.team_id = teams.id*

This sentence says: "Show me the name and age of all players who are older than 18, and also show me their team information."

---

**SQL can do four main things:**

- **Create** new data
- **Read** existing data
- **Update** data
- **Delete** data

We call this **CRUD** — Create, Read, Update, Delete.

---

**That's it! Let's review.**

- SQL is a language for databases.
- A database stores data in tables with rows and columns.
- Each row has a primary key.
- We use foreign keys to connect tables.
- We use SELECT, FROM, WHERE, and JOIN to read data.

SQL is used by millions of developers around the world. If you want to work with data, learning SQL is a great first step. Good luck!
